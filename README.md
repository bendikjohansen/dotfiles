# Dotfiles

## Components

### Alacritty

[Alacritty](https://github.com/alacritty/alacritty) is a Rust-based terminal that supports true color mode.
It is very simple and configurable through a dot-file.

### Git

#### delta
[delta](https://github.com/dandavison/delta) is used for clearer git diffs.

### Tmux
[Tmux](https://github.com/tmux/tmux/wiki) is a terminal multiplexer.
This means you can split your window into panes, have multiple windows and start
multiple sessions.

### Starship
[Starship](https://starship.rs/) is the prompt for the terminal. It is powered
by rust and features a very nice, minimal prompt out of the box.
