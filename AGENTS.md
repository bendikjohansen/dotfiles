# dotfiles

A collection of dotfiles to configure personal environments.

## Instructions

- When on macOS, only consider the `./mac` and `./bin` folders unless explicitly told otherwise.
- When on Linux, only consider the `./linux` and `./bin` folders unless explicitly told otherwise.

## Systems

### MacBook Pro

- *Specifications*: Apple M4, 32 GB RAM, 512 GB SSD, Tahoe 26.3.1
- *dotfiles location*: `./mac/`

### Linux

- Currently not running on any machine.
- dotfiles location: `./linux/`
