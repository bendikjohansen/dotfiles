# Mac

## Setup

1. Navigate to the `./mac/` directory.
2. Execute `./setup.sh`

## Philosophy

This Mac configuration aims to be simple and powerful. It often relies often
sensible defaults.

In some cases, it prefers "good enough" to "perfect" if it means keeping the
system dependencies fewer.
Two such examples are the terminal and the window managers, which both are
offered in the stock application suite.
